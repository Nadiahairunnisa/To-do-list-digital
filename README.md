# To-Do List Digital berbasis web untuk manajemen tugas harian. Dibuat sebagai project akhir SMK NEGERI 11 PANDEGLANG jurusan Rekayasa Perangkat Lunak
## Teknologi:                                                                                                                                   -Backend : PHP Native                                                                                                                       -Database : MySQL                                                                                                                           -Server : XAMPP                                                                                                                             -Editor : Dreamweaver
## Fitur utama:                                                                                                                                 -Autentikasi : sistem login & logout user                                                                                                   -CRUD Tugas : tambah, lihat, ubah, hapus to do list                                                                                         -Responsive : bisa dibuka di laptop
## cara menjalankan 
1. Jalankan XAMPP > Start Apache & MySQL
2. Buka localhost/todo-list di browser
3. Login default Username & Password
